1. IOT Framework

Basic constituency of an IOT framework consist of device layer, connectivity layer and computational layer.

2. IOT Framework - Device layer

Devices layer is the thing in the 'internet of things'. It can be anything from manufacturing robots to lights in your house. All these things contain embedded devices with one or more sensors for gathering data, processors to convert this data to machine understandable format either to transmit it or to take decision at the site.

3. IOT Framework - Communication layer

Communication layer is the internet in the 'internet of things'. communication devices are found in the 'things' can be can be connected to the internet via a local hub or directly. Mutliple medium of communication can be used like wireless or wired mediums to transmit the gathered data to the IOT cloud for further process. We will take an in depth look at these IOT communication protocols later in the course.

4. IOT Framework - Computational layer

The information gathered from the IOT system is stored in the cloud and we can use this data to numerous activity. Thats where computatoinal layer comes in the picture. We can run analytics and derive infomation which can help us derive conclusion, perform predictive maintance.
